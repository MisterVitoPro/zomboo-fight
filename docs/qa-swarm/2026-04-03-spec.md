## P0 Fixes (Implementation-Ready)

### P0-005: Initialize donePlaying before menu loop

**File:** `gameLib/main.py`

**Before:**
```python
def menu():
    screen = pygame.display.set_mode((800, 600))
    pygame.display.set_caption("ZOMBOO")
    
    background = pygame.image.load(dataFiles.mainMenuIm)
    screen.blit(background, (0, 0))
    
    keepGoing = 1
    clock = pygame.time.Clock()
    pygame.mouse.set_visible(False)
    while keepGoing == 1:
        background = pygame.image.load(dataFiles.mainMenuIm)
```

**After:**
```python
def menu():
    screen = pygame.display.set_mode((800, 600))
    pygame.display.set_caption("ZOMBOO")
    
    background = pygame.image.load(dataFiles.mainMenuIm)
    screen.blit(background, (0, 0))
    
    keepGoing = 1
    donePlaying = True
    clock = pygame.time.Clock()
    pygame.mouse.set_visible(False)
    while keepGoing == 1:
```

**Rationale:** If the loop exits without an event (display closed externally), `donePlaying` defaults to `True` (quit), preventing both the `UnboundLocalError` and an accidental game start.

### P0-002: Clamp spawn rate minimums

**File:** `gameLib/main.py`

**Before:**
```python
                zombieSMin -= 6
                zombieSMax -= 6
```

**After:**
```python
                zombieSMin = max(zombieSMin - 6, 5)
                zombieSMax = max(zombieSMax - 6, 10)
```

**Rationale:** Prevents `zombieSMin` and `zombieSMax` from going to zero or negative, which would crash `random.randrange()`. Floor values of 5 and 10 keep a minimum spread and ensure valid range arguments.

### P0-003: Fix BigZombie spawn range

**File:** `gameLib/main.py`

**Before:**
```python
    def spawnZombie(self, pos):
        zNum = random.randrange(1, 12)
        if zNum <= 11:
            zombie = sprites.Zombie(dataFiles.zombie1, pos)
            sGroups.zombieSprites.add(zombie)
        elif zNum > 13:
            bigZombie = sprites.BigZombie(dataFiles.zombieBig, pos)
            sGroups.zombieSprites.add(bigZombie)
```

**After:**
```python
    def spawnZombie(self, pos):
        zNum = random.randrange(1, 13)
        if zNum <= 11:
            zombie = sprites.Zombie(dataFiles.zombie1, pos)
            sGroups.zombieSprites.add(zombie)
        else:
            bigZombie = sprites.BigZombie(dataFiles.zombieBig, pos)
            sGroups.zombieSprites.add(bigZombie)
```

**Rationale:** `randrange(1, 13)` gives 1-12. Values 1-11 spawn normal zombies (~92%), value 12 spawns BigZombie (~8%). Using `else` instead of `elif zNum > 13` ensures the branch is reachable.

## P1 Fixes (Strategic)

for group_name in ['turretSprites', 'UISprites', 'UIBarSprites', 'textSprites',
