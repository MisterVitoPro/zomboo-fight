## Test Infrastructure

This project has no existing test infrastructure. Tests require:

1. **pytest** as test runner
2. **pygame mock/stub** -- pygame requires a display; tests should use `pygame.display.set_mode()` with a dummy video driver:
   ```python
   import os
   os.environ['SDL_VIDEODRIVER'] = 'dummy'
   os.environ['SDL_AUDIODRIVER'] = 'dummy'
   ```
3. Tests go in `tests/` directory at project root
4. A `conftest.py` to handle pygame init/quit
